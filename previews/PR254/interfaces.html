Module Interfaces

Rimu.Interfaces — Module
module Interfaces

This module contains interfaces that can be used to extend and modify the algorithms and behaviours of Rimu.

Interfaces

Follow the links for the definitions of the interfaces!

  • AbstractHamiltonian for defining Hamiltonians
  • AbstractDVec for defining data structures for Rimu as in DictVectors
  • StochasticStyle for controlling the stochastic algorithms used by lomc! as implemented in StochasticStyles
